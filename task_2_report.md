# **Task 2: Data Visualization using the Iris Dataset**

## **1. Introduction**

Data visualization is an important stage in data analysis that helps identify patterns, trends, and relationships within a dataset. This project explores the famous Iris dataset using various visualization techniques to better understand the characteristics of different Iris flower species.

---

## **2. Objective**

The objectives of this project are:

* To load and explore the Iris dataset.
* To visualize feature distributions using graphical methods.
* To compare relationships among different features.
* To identify patterns among the three Iris species.

---

## **3. Dataset Description**

The Iris dataset is one of the most widely used datasets in machine learning.

* Dataset Name: Iris Dataset
* Number of Samples: 150
* Number of Features: 4
* Target Classes:

  * Iris Setosa
  * Iris Versicolor
  * Iris Virginica

The four numerical features are:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

---

## **4. Methodology**

The following steps were performed during the project:

1. Imported the required Python libraries.
2. Loaded the Iris dataset using Scikit-learn.
3. Converted the dataset into a Pandas DataFrame.
4. Performed exploratory data analysis.
5. Generated different visualizations including:

   * Bar Chart
   * Histogram
   * Scatter Plot
   * Pair Plot
   * Correlation Heatmap

---

## **5. Results and Discussion**

The visualizations provided several insights:

* The dataset contains an equal number of samples for each species.
* Petal Length and Petal Width have a strong positive correlation.
* Iris Setosa is clearly separated from the other two species based on petal measurements.
* Versicolor and Virginica show some overlap but remain distinguishable using multiple features.
* Histograms illustrate the distribution and variability of each feature.

These observations demonstrate the usefulness of visualization for understanding data before building predictive models.

---

## **6. Conclusion**

The project successfully demonstrated various visualization techniques for the Iris dataset. The generated charts effectively highlighted feature distributions and relationships among species, making the dataset easier to interpret. Exploratory data analysis provides valuable insights that serve as a strong foundation for future machine learning applications.
